Coloque aqui suas imagens do site.

- Acesse via URL: `/images/NOME_DO_ARQUIVO.ext`.
- Em código React, use: `url(${process.env.PUBLIC_URL}/images/NOME_DO_ARQUIVO.ext)`.

Exemplo para o fundo da página Contato:
- Arquivo: `public/images/contato-bg.png`
- Uso: `url(${process.env.PUBLIC_URL}/images/contato-bg.png)`
